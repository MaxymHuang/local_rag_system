How to Use

1. Initialize: Enter a directory path and click "Initialize" to scan files
2. Search: Type keywords to find relevant files in your directory
3. Summarize: Click "Summarize" on any file to get an AI-generated summary (can be disabled in settings)
4. Chat: Use the chat box to ask questions or get help from Ollama AI
5. Settings: Configure Ollama server, model, and toggle AI features on/off

Tip: Make sure Ollama is running before using AI features!

About This Project

This File Finder RAG System was built originally for solving a headache at work. There was a huge network file system at work that was a pain to navigate, and everything was poorly organized. I thought it would be nice to have a way to search for files across the network without having to know the exact path and without having to search through folders and folders of documents. From there, I thought it would be even better if this could connect to an AI agent that summarizes the files for me, that way I don't even have to open the file to get the gist of it.

Built with Python, Flask, and AI generation (I'm not even going to lie to you)
Powered by Ollama for local AI processing
Uses semantic search for intelligent file discovery
Features a modern, responsive design
